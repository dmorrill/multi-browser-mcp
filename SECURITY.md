# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Multi-Browser MCP, please report it responsibly.

**Email:** danielle@firstpartyhq.com

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within 48 hours and provide a timeline for a fix.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.9.x   | ✅ Yes    |
| < 1.9   | ❌ No     |
